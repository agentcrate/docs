---

**AgentCrate** brings the Docker workflow to AI agents. Define your agent in an
`Agentfile`, build it into a signed OCI artifact, and distribute it through
**CrateHub** — the registry — or any OCI-compatible registry of your choice.

## Two things, one platform

- **AgentCrate** — the platform: SDKs, CLI, signing, vetting, runtime.
- **CrateHub** — the registry: public discovery, private namespaces, install metrics.

If you've used Docker and Docker Hub, the relationship is the same.
